Assignment 3: Interactive Games

This assignment demonstrates interactive web development using HTML5, CSS3, and JavaScript. Two engaging games showcase different programming concepts and user interface design principles.

HTML5 CSS3 JavaScript ES6 Local Storage Responsive Design

Number Guessing Game

Challenge yourself to guess the secret number between 1 and 100! You have 10 attempts to find the correct number. The game includes a timer, sound effects, and helpful hints to guide your guesses.

Features: Timer tracking, audio feedback, automatic game restart, input validation, and responsive design.

Memory Matching Game

Test your memory with this challenging card matching game. Choose your difficulty level and number of pairs, then try to match all cards before time runs out.

Features: Multiple difficulty levels, leaderboard system, memorization phase, score tracking, and beautiful card animations.
