Project Overview

A fully-featured web-based Jeopardy game that brings the classic TV show experience to your browser. Built with PHP, MySQL-style file storage, and modern responsive design, this project demonstrates comprehensive full-stack web development skills including user authentication, session management, persistent data storage, and dynamic content generation.

🔐

User Authentication

Secure registration and login system with password hashing, session management, and "Remember Me" functionality. Includes comprehensive input validation and security measures.

🎯

Dynamic Game Logic

Randomized question selection from multiple question banks per category, real-time score tracking, and persistent game state management across sessions.

💾

File-Based Storage

Persistent user data and score storage using secure file operations with proper permissions and data validation. No database required - pure PHP file handling.

🏆

Leaderboard System

Global scoring system with persistent rankings, user identification, and competitive gameplay features. Tracks top performers across all game sessions.

📱

Responsive Design

Mobile-first responsive design with CSS animations, hover effects, and adaptive layouts. Includes the iconic Jeopardy blue and gold color scheme.

⚡

Real-Time Feedback

Instant answer validation, visual feedback for correct/incorrect responses, and smooth transitions between game states with CSS animations.
